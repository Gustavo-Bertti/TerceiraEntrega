#  Mobile App – Documentação Técnica

##  Visão Geral
Aplicação mobile desenvolvida com **React Native + Expo + TypeScript**, seguindo uma arquitetura modular para facilitar manutenção, escalabilidade e reutilização de código.

---

##  Tecnologias
- React Native
- Expo
- TypeScript
- Expo Router
- Hooks personalizados
- Tema Dark / Light

---

##  Arquitetura

- **app/**: ponto de entrada da aplicação
- **components/**: componentes reutilizáveis
- **Screens/**: telas da aplicação
- **config/**: configuração de API e autenticação
- **constants/**: constantes globais e temas
- **hooks/**: hooks personalizados
- **routes/**: controle de navegação
- **types/**: tipagens e DTOs

---

##  Setup Inicial

### Pré-requisitos
- Node.js >= 18
- Expo CLI
- Android Studio
